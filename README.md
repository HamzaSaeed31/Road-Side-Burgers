## 📖 About

A multi-page front-end website for **Road-Side Burgers**, a fictional fast-food brand. Built as an ICT Lab project, it demonstrates core web development skills including responsive layout, dynamic cart logic and form handling — all in vanilla HTML, CSS and JavaScript.

## ✨ Features

- 🍔 **Menu Page** — Categorised food items with prices and images
- 🛒 **Shopping Cart** — Add/remove items, quantity control, live total
- 🎉 **Deals Page** — Promotional offers and combo meals
- 💳 **Payment Page** — Checkout form with order summary
- 🔐 **Login / Sign-Up** — User authentication forms

## 🗂️ Pages

| File | Description |
|---|---|
| `main.html` | Homepage / landing |
| `menu.html` | Food menu |
| `cart.html` | Shopping cart |
| `deals.html` | Current deals |
| `payment.html` | Checkout |
| `login.html` | Login & registration |

## 🚀 Getting Started

No build tools required — just open in a browser.

```bash
# Clone and open
git clone https://github.com/HamzaSaeed31/Road-Side-Burgers.git
cd Road-Side-Burgers
open main.html   # or double-click in file explorer
```
